# Guide Utilisateur - LE SCRIPT

## 📱 Pour les Artistes

### 1. Créer un Compte Artiste
1. Aller sur LE SCRIPT
2. Cliquer sur "S'inscrire"
3. Remplir le formulaire avec:
   - Email
   - Mot de passe (min 8 caractères)
   - Sélectionner "Artiste" comme rôle
4. Cliquer sur "S'inscrire"

### 2. Compléter votre Profil
1. Aller à "Profil"
2. Remplir les informations:
   - Nom d'artiste 🎤
   - Catégorie musicale (Hip-Hop, Pop, Rock, etc.)
   - Bio
   - Photo de profil
3. Cliquer "Sauvegarder"

### 3. Créer un Projet
1. Aller à "Mes Projets"
2. Cliquer sur "+ Nouveau Projet"
3. Remplir les détails:
   - Titre du projet
   - Pochette/Cover (image)
4. Cliquer "Créer"

### 4. Ajouter des Liens Streaming
1. Ouvrir votre projet
2. Cliquer sur "Ajouter un lien"
3. Sélectionner la plateforme:
   - 🎵 Spotify
   - 🍎 Apple Music
   - 📺 YouTube Music
   - ☁️ SoundCloud
   - 🎼 Bandcamp
   - Et bien d'autres...
4. Coller le lien URL
5. Le logo s'affichera automatiquement! 🎉

### 5. Suivre vos Certifications
Votre projet gagne des points automatiquement quand les certifieurs réagissent:

**Seuils de Certification:**
- 🥉 Bronze: 50 points
- 🥇 Or: 250 points
- 🥈 Platine: 500 points
- 💎 Diamant: 1000 points + badge bleu ✅

Quand vous atteignez un seuil:
- ✉️ Certificat envoyé par email
- 📊 Badge affiché sur votre profil
- 🎊 Notification de la communauté

## ⭐ Pour les Certifieurs

### 1. Créer un Compte Certifieur
1. Aller sur LE SCRIPT
2. Cliquer sur "S'inscrire"
3. Remplir le formulaire:
   - Email
   - Mot de passe
   - Sélectionner "Certifieur"
4. Cliquer "S'inscrire"

### 2. Compléter votre Profil
1. Aller à "Mon Profil"
2. Ajouter vos infos:
   - Prénom
   - Nom
   - Genre (optionnel)
3. Sauvegarder

### 3. Voter sur les Projets
1. Aller à "Découvrir les Projets"
2. Parcourir les projets musicaux
3. Pour chaque projet, vous pouvez:
   - ❤️ **Like** (+1 point)
   - 👎 **Dislike** (-1 point)
   - 💬 **Commenter** (+2 points)
   - 🔄 **Partager** (+4 points)
   - 🎰 **Joker** (50 points, payant 2€)

### 4. Gagner des Badges
Vos réactions sont comptabilisées:

**Paliers de Badges:**
- ⭐ Or: 100 réactions
- 🟢 Vert: 200 réactions
- 🟣 Violet: 500 réactions

Vos badges apparaîtront sous votre nom! 🌟

### 5. Utiliser les Jokers
Les Jokers permettent de donner 50 points d'un coup (premium):

1. Aller sur un projet
2. Cliquer sur "🎰 Utiliser un Joker"
3. Payer 2€ avec Stripe
4. +50 points automatiquement ajoutés! 🎉

## 🔒 Sécurité & Confidentialité

### Protéger votre Compte
- ✅ Utilisez un mot de passe fort (min 8 caractères)
- ✅ Ne partagez pas votre token JWT
- ✅ Vérifiez votre email après inscription
- ✅ Mettez à jour votre profil régulièrement

### Paiements Sécurisés
- 🔒 Stripe gère tous les paiements
- 💳 Vos données bancaires ne sont jamais stockées
- 🛡️ Chiffrement SSL/TLS

## 🆘 Problèmes Courants

### Je n'arrive pas à me connecter
1. Vérifiez votre email et mot de passe
2. Réinitialisez votre mot de passe
3. Videz le cache de votre navigateur

### Mon projet n'apparaît pas
1. Vérifiez que tous les champs sont remplis
2. La page se met à jour en temps réel
3. Rechargez la page si nécessaire

### Le paiement du Joker a échoué
1. Vérifiez votre connexion internet
2. Essayez une autre carte
3. Contactez votre banque

### Je n'ai pas reçu mon certificat
1. Vérifiez vos spams
2. Vérifiez que votre email est correct
3. Contactez le support
